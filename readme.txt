== Description ==

Elevate your WordPress editing experience with the Text Popover plugin, designed to seamlessly integrate with various blocks featuring toolbars such as paragraph blocks, heading blocks, and more. Enjoy the flexibility of adding popover content to selected text, enhancing your content creation process.

= Special Features =

* Effortlessly add popovers to selected text within WordPress blocks.

== Installation ==

1. Upload the entire `Text Popover` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= Minimum Requirements =

* WordPress 5.0 or higher

= Manual Installation =

1. Unzip the files and upload the folder into your plugins folder (/wp-content/plugins/) overwriting older versions if they exist
2. Activate the plugin in your WordPress admin area.
